Documentation

C'est un template qui permet de construire des sites web rapidement avec des codes réutilisables JS et CSS, mais en laissant la flexibilité aux développeurs front-end le plus possible. Je travaille ou je planifie de travailler sur plusieurs versions, dont celui-ci est la version avec Gulp qui compile le HTML, le CSS et JS en toute beauté.

Ici, vous pouvez configurer les valeurs du design system aux variables contenus dans le fichier « settings.scss » affectant les espacements, les palettes de couleurs et la typographie. Ça aide à garder l'uniformité de design le plus possible.

Installation

Téléchargez le répertoire GitHub selon la version de votre choix. Les utilisateurs de GitHub peuvent créer un nouveau répertoire en cliquant sur le bouton « Use this template ».

  • Gulp
  • Vite
  • Astro
  • Qwik

Ensuite, ouvrez le terminal (ou par l'intermédiaire de l'éditeur de texte comme VS Code) au dossier du projet. Écrivez dans le terminal cette commande :

Avec NPM :

$ npm i

Commencer la conception

Avec NPM :

$ npm start
  //ou
  $ npm run dev

Compiler le site web

Avec NPM :

$ npm run build
